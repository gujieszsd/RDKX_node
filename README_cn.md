# 基于RDK X5的茶树巡护动态跟踪激光灭虫机器人
## 项目简介
本项目以地平线RDK X5为边缘计算核心，实现茶园自主巡行、害虫AI识别、云台动态跟踪、激光靶向物理灭虫，无农药残留。
## 硬件清单
地平线RDK X5开发板、D435深度相机、MPU6050六轴传感器、CAN总线步进电机、808nm激光模组、二维跟踪云台
## 代码结构
- motor_controller_node.py：电机运动控制ROS节点
- mpu6050_angle_publisher.py：姿态传感器数据发布节点
## 部署环境
Ubuntu 容器化部署、ROS、轻量化YOLO害虫检测算法
## 运行说明
1. 启动传感器节点
2. 启动电机控制节点
3. 启动AI识别与云台跟踪推理程序
