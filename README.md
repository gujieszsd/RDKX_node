# Tea Tree Patrol & Dynamic Tracking Laser Precision Target Pest Control Robot Based on RDK X5
## Project Overview
This robot takes Horizon RDK X5 edge computing board as the core controller, aiming at physical pesticide-free pest control for mountain tea gardens.
It realizes full autonomous functions including tea ridge patrol, real-time pest identification via lightweight YOLO algorithm, Kalman filter trajectory prediction, 2-DOF gimbal dynamic tracking, and adjustable-power laser targeted insect elimination.
The system adopts containerized Ubuntu deployment, supports SLAM mapping, remote VNC debugging, CAN bus multi-motor coordinated drive and MPU6050 attitude sensing, which can replace manual work to reduce pesticide residues in organic tea planting.

## Hardware List
- Horizon RDK X5 Development Board
- Intel RealSense D435 Depth Camera
- MPU6050 6-axis IMU Attitude Sensor
- CAN FD Bus Stepper Motor Drive System
- 2-DOF Servo Tracking Gimbal
- 808nm Adjustable Power Laser Module
- Lithium Battery Power Supply System
